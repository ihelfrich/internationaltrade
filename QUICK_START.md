# Quick Start

Open the teaching hub first:
- `index.html`

Key destinations:
- Slides: `slides.html`
- Interactive lab: `hub.html`
- Lectures: `lectures/ricardian.html`, `lectures/specific_factors.html`
- Notes: `notes/ricardian.md`, `notes/specific-factors.md`, `notes/gravity.md`

Recommended flow:
1. Teach from the lecture page.
2. Use slides for visuals and pacing.
3. Demonstrate interactives in the lab.
4. Assign the notes.
